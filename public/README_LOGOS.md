# 로고 파일 넣는 법 💜

이 폴더에 아래 두 파일을 넣어주세요:

1. **bts-logo.png** — BTS 공식 로고 (투명 배경 PNG 또는 SVG)
2. **army-logo.png** — ARMY 공식 로고 (투명 배경 PNG 또는 SVG)

## 로고 다운로드

- 구글에서 "BTS logo transparent PNG" 검색
- 구글에서 "BTS ARMY logo transparent PNG" 검색
- 또는 위키피디아에서 SVG 다운로드: https://en.wikipedia.org/wiki/File:BTS_logo.svg

## 참고사항

- 투명 배경(transparent) PNG를 권장합니다
- 흰색 로고가 가장 잘 어울립니다 (다크 배경이므로)
- 흰색 로고가 없으면 코드에서 자동으로 흰색으로 변환합니다
- 파일이 없으면 "BTS" / "ARMY" 텍스트로 자동 대체됩니다
